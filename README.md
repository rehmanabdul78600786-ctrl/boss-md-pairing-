# 🚀 BOSS MD - Premium WhatsApp Pairing Website

A modern, feature-rich WhatsApp pairing website for BOSS MD bot.

## ✨ Features

- 🔗 **Dual Pairing Methods** - Pair Code & QR Code
- 📱 **Auto WhatsApp Messages** - Welcome message sent automatically
- 👑 **Owner Notifications** - Session details sent to owner
- 🎨 **Premium Design** - Neon dark theme with animations
- 📊 **Live Status** - Server status, active sessions
- 🔒 **Secure** - Session expiry, request validation
- 📱 **Fully Responsive** - Works on all devices

## 📋 Prerequisites

- Node.js 18+
- npm or yarn
- WhatsApp account

## 🛠️ Installation

```bash
# Clone repository
git clone https://github.com/your-username/boss-md-pairing.git
cd boss-md-pairing

# Install dependencies
npm install

# Configure environment
cp backend/.env.example backend/.env
# Edit .env with your settings

# Start server
npm start
